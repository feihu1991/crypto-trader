# CryptoQuant - 数字货币量化交易工具

一个简洁的数字货币量化交易网页应用，支持模拟交易和实盘交易。

![Preview](https://img.shields.io/badge/HTML5-CSS3-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 功能特性

### 📊 实时行情
- 显示主流币种实时价格
- 24小时涨跌幅
- 自动刷新

### 💰 交易功能
- 买入/卖出操作
- 持仓管理
- 盈亏计算
- 交易历史记录

### 🎯 策略交易
- 可视化策略配置
- 条件触发自动交易
- 策略保存与加载

### 🏓 模拟盘 (Paper Trading)
- 虚拟资金
- 无风险练习
- 实时模拟交易

### 🔄 实盘交易
- 支持 Binance API
- 真实资金交易
- ⚠️ 需要API密钥

## 使用说明

### 1. 打开应用
直接在浏览器中打开 `index.html`

### 2. 基本操作
- 点击价格旁边的"买入/卖出"按钮进行交易
- 在"设置"中配置初始资金

### 3. 策略配置
1. 选择触发币种（BTC/ETH等）
2. 设置条件（涨幅超过/跌幅超过）
3. 设置数值（百分比）
4. 选择操作（买入/卖出）
5. 设置金额
6. 点击"保存"

### 4. 切换实盘
1. 点击"设置"
2. 选择"实盘交易"
3. 输入 Binance API Key 和 Secret Key
4. ⚠️ 注意：API密钥仅保存在浏览器本地

## 技术栈

- HTML5 + CSS3
- Vanilla JavaScript
- Binance Public API
